# 🔊 VOICE — テキスト読み上げアプリ

ブラウザで動く、オフライン対応のテキスト読み上げアプリです。  
**PWA対応**により、スマホ・PCのホーム画面にインストールして、ネイティブアプリのように使えます。

---

## ✨ 機能

- **日本語 / 英語** の切り替え対応
- **読み上げ中の単語をリアルタイムハイライト**
- 読み上げ速度・音程・音量の調整
- OSにインストール済みの音声エンジンから選択可能
- テキストエリア右下の再生ボタン（スマホでもすぐ操作できる）
- 読み上げ完了後、テキスト入力欄の末尾にカーソルが自動移動
- 豊富なキーボードショートカット
- **PWA対応** — ホーム画面にインストール・オフライン動作

---

## 🚀 使い方

### ブラウザで使う（ファイルを直接開く）

1. `tts-app.html` をダウンロード
2. Chrome または Edge でファイルを開く
3. テキストを入力して再生ボタンを押す

### PWAとしてインストールする（推奨）

GitHub Pages などで公開されたURLにアクセスして、ホーム画面に追加します。

**Android (Chrome)**
→ サイトを開く → メニュー「ホーム画面に追加」

**iPhone (Safari)**
→ サイトを開く → 共有ボタン →「ホーム画面に追加」

**PC (Chrome / Edge)**
→ アドレスバー右端のインストールアイコンをクリック

> **オフライン動作について**  
> Chrome / Edge はOSの音声エンジンを使用するため、インターネット接続なしで動作します。  
> Firefox は一部オンライン取得が必要な場合があります。

## ⌨️ キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl + Enter` | 再生 / 一時停止 |
| `Escape` | 停止 |
| `Ctrl + →` / `Ctrl + l` | 読み上げ速度 +0.1x |
| `Ctrl + ←` / `Ctrl + h` | 読み上げ速度 -0.1x |
| `Ctrl + ↑` / `Ctrl + k` | 音量 +5% |
| `Ctrl + ↓` / `Ctrl + j` | 音量 -5% |
| `Space` | 再生 / 一時停止（テキスト入力欄外） |

> `Ctrl + hjkl` はテキスト入力中でも動作します。

## 📁 ファイル構成

```
voice-tts-app/
├── tts-app.html    # アプリ本体
├── manifest.json   # PWA設定（アプリ名・アイコン・テーマ）
├── sw.js           # Service Worker（オフラインキャッシュ）
├── icon-192.png    # ホーム画面アイコン（192×192）
├── icon-512.png    # ホーム画面アイコン（512×512）
└── README.md
```

> PWAはHTTPS環境が必要です。GitHub Pages はデフォルトでHTTPS対応しています。

## 🛠 GitHub Pages での公開手順

1. このリポジトリを GitHub に作成・アップロード
2. Settings → Pages → Branch: `main` / `/ (root)` を選択して Save
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/tts-app.html` で公開される
