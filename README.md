# 🔊 VOICE — テキスト読み上げアプリ

ブラウザで動く、シンプルな読み上げアプリです。  
単一HTMLファイルで完結しており、**インストール不要・オフライン対応**です。

---

## ✨ 機能

- **日本語 / 英語** の切り替え対応
- **読み上げ中の単語をリアルタイムハイライト**
- 読み上げ速度・音程・音量の調整
- OSにインストール済みの音声エンジンから選択可能
- 読み上げ完了後、テキスト入力欄の末尾にカーソルが自動移動
- 豊富なキーボードショートカット

---

## 🚀 使い方

1. `tts-app.html` をダウンロード
2. Chrome または Edge でファイルを開く
3. テキストを入力して再生ボタンを押す

> **オフライン動作について**  
> Chrome / Edge はOSの音声エンジンを使用するため、インターネット接続なしで動作します。  
> Firefox は一部オンライン取得が必要な場合があります。

---

## ⌨️ キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl + Enter` | 再生 / 一時停止 |
| `Escape` | 停止 |
| `Ctrl + →` / `Ctrl + l` | 読み上げ速度 +0.1x |
| `Ctrl + ←` / `Ctrl + h` | 読み上げ速度 -0.1x |
| `Ctrl + ↑` / `Ctrl + k` | 音量 +5% |
| `Ctrl + ↓` / `Ctrl + j` | 音量 -5% |
| `Space` | 再生 / 一時停止（テキスト入力欄外） |

> `Ctrl + hjkl` はテキスト入力中でも動作します。
